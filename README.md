# 🛒 E-Commerce Rookie Training Project

This repository contains an **E-Commerce application** developed as part of the **Rookie Training Program at NashTech**.

## 📌 Overview

The project is designed to help trainees understand and apply modern software development practices using the **.NET ecosystem**, including:

* Clean and scalable architecture (Vertical Slice Architecture)
* RESTful API design
* Entity Framework Core for data access
* Dependency Injection & middleware pipeline
* Authentication & Authorization (JWT / Identity)
* Best practices in code organization and maintainability

## 🎯 Objectives

* Gain hands-on experience building real-world applications
* Apply backend development concepts in a structured environment
* Understand enterprise-level coding standards and architecture

## 🧱 Tech Stack

* ASP.NET Core Web API
* Razor Pages
* Entity Framework Core
* SQL Server
* Duende Identity Server

## 👨‍💻 Author

Developed during the **Rookie Program at NashTech** as part of backend training.
